PROJECTS / OCTAVIUS

WHAT?


A tangible performance platform for electronic music. Using basic interaction to compose, perform and manipulate sound.

Each device is wireless and uses its orientation to manipulate parameters whilst each pad can trigger sounds or events.

OCTAVIUS explores basic interactions and new technology to offer a different kind of experience with sound. It uses basic interactions that musicians are familiar with to offer a different, expressive experience.

TECH SPEC

OCTAVIUS uses mesh networking and sensors to offer up a wireless and tangible experience. Each of the eight blocks can be rotated independently to manipulate effects and parameters using the MIDI and OSC protocols.

SOFTWARE

Using software built in MAX, OCTAVIUS can change different instruments seamlessly on the fly. With assignable orientation parameters for each device. Works with both hardware and software.
